# Fewd2 Static Site

Simple static front-end project containing multiple HTML pages (login, signup, timetable, etc.).

## Contents
- `index.html` – Landing page
- `home.html` – Home/dashboard
- Additional HTML pages (login, signin, settings, timetable, attendance, announcements, help)
- CSS files for styling
- `home.js` for any interactive logic

## Getting Started
Open `index.html` or `home.html` directly in a browser.

You can add more JS/CSS and commit changes using Git.
